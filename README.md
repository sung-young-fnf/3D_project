# F&F STORE - VMD 프로토타입

Spark 기반 3D 가상 매장 디스플레이 테스트

---

## 실행 방법

### 1. 터미널에서 spark 폴더로 이동
```bash
cd C:\Users\AC0833\project\3D_project\spark
```

### 2. 의존성 설치 (최초 1회만)
```bash
npm install
```

### 3. 개발 서버 실행
```bash
npx vite --port 8080
```

### 4. 브라우저에서 접속
```
http://localhost:8080/examples/vmd-test/
http://localhost:8080/examples/vmd-test2/
```

### 5. 서버 끄기
터미널에서 `Ctrl + C`
